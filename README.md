# E-commerce Website

A modern, responsive e-commerce website built with HTML, CSS, and JavaScript.

## Features

- **Responsive Design**: Works on all devices (desktop, tablet, mobile)
- **Product Catalog**: Browse products by categories
- **Shopping Cart**: Add/remove items, update quantities
- **User Authentication**: Secure login and registration
- **Product Search**: Find products quickly with the search functionality
- **Secure Checkout**: Safe and secure payment processing

## Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Icons**: Font Awesome 6
- **Version Control**: Git & GitHub

## Getting Started

### Prerequisites

- Modern web browser (Chrome, Firefox, Safari, Edge)
- Git (for development)
- Code editor (VS Code, Sublime Text, etc.)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Tusharsingh7705/Ecommerce_web.git
   cd Ecommerce_web
   ```

2. Open `index.html` in your preferred web browser.

## Development

### Running Locally

1. Open the project folder in your code editor
2. Use a live server extension (e.g., Live Server in VS Code) to run the project
3. The application will open in your default browser

### Building for Production

This project is built with vanilla web technologies, so no build step is required. However, for production, you might want to:

1. Minify CSS and JavaScript
2. Optimize images
3. Enable GZIP compression on your web server
